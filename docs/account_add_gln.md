# Módulo: Add Partner GLN

## Información General
- **Nombre técnico:** account_add_gln
- **Versión:** 1.0
- **Categoría:** Accounting/Accounting
- **Dependencias:** account


## Propósito
This module adds the Global Location Number to the partner. Used on delivery addresses, it is used to identify stock locations and is mandatory on the UBL/CII eInvoices (but not only). The module is intended be merged with account, later on, in master
