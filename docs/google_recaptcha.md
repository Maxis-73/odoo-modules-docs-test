# Módulo: Google reCAPTCHA integration

## Información General
- **Nombre técnico:** google_recaptcha
- **Versión:** 1.0
- **Categoría:** Hidden
- **Dependencias:** base_setup




## Descripción

This module implements reCaptchaV3 so that you can prevent bot spam on your public modules.
    



## Modelos

### res.config.settings


- Hereda de:

  - res.config.settings




#### Campos
- **recaptcha_public_key** (Char) → Site Key
- **recaptcha_private_key** (Char) → Secret Key
- **recaptcha_min_score** (Float) → Minimum score
